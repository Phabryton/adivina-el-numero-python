# Adivina el Número Secreto 🎮

¡Mi primer proyecto en Python! Un divertido juego de consola donde tienes que adivinar un número aleatorio entre 1 y 10.

## 🚀 Características
* Generación de números aleatorios con la librería `random`.
* Sistema de control de intentos (máximo 3 vidas).
* Pistas en tiempo real ("sube más" o "baja más") para orientar al jugador.
* Estructurado completamente con bucles `while` y condicionales `if/elif/else`.
